PyPI mirror · cache · private index

Serve Python packages in milliseconds.

velox is a PyPI-compatible read-through cache and private index in one small Rust binary. Point pip, uv, or twine at it; the only change a client needs is the index URL.

Get started Source on GitHub 
$ velox serve
velox listening on 127.0.0.1:4433 · 3 indexes
